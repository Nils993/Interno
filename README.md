# Проект: Каталог дизайна интерьеров

Суть проекта: Создание многостраничного сайта-каталога, демонстрирующего различные дизайн-проекты интерьеров. В рамках проекта реализованы интерактивные компоненты и навигация для удобного просмотра и фильтрации дизайнов.

Что удалось сделать:
-	Вёрстка всех страниц полностью выполнена по макету из Figma
-	Реализована маршрутизация с использованием Vue Router, включая обработку 404 ошибок
-	Созданы компоненты пагинации и фильтров, обеспечивающие удобство навигации по контенту
-	Настроено управление состоянием через Vuex для хранения статей, карточек и слайдов

Стек:
- Vue.js,
- Vuex,
- Vue Router,
- CSS3 (Flexbox/Grid),
- JavaScript


# Установка и запуск проекта
   Сначала откройте терминал и выполните команду:
- `npm install `
Для запуска проекта в режими разработчика введите команду в консоли
- `npm run serve`
  Теперь вы сможете открыть проект в браузере по адресу `http://localhost:8080` (или другому, указанному в терминале).

Для настройки конфигурации проекта вы можете ознакомиться с [документацией](https://cli.vuejs.org/config/).
